# 🛠️ TechConnect - Marketplace SaaS

Plataforma SaaS geolocalizada para conectar clientes a técnicos especializados. Desenvolvido para facilitar orçamentos e visibilidade profissional.

---

### 🚀 Tecnologias
* **Linguagem:** Java 17+ (Spring Boot 3)
* **Banco de Dados:** PostgreSQL + PostGIS (Geolocalização)
* **Segurança:** Spring Security + JWT
* **Documentação:** Swagger/OpenAPI
* **Arquitetura:** REST API com Maven

---

### 📋 Principais Funcionalidades
* 📍 **Mapa Interativo:** Busca de técnicos por proximidade.
* 🔍 **Filtros:** Especialidades (Elétrica, TI, etc.) e avaliações.
* 💰 **Orçamentos:** Solicitação e acompanhamento de status online.
* 💎 **Modelo SaaS:** Planos Premium para destaque de técnicos.
* 🔐 **Segurança:** Login seguro para clientes e profissionais.
